Package Overview
KA.Barcode.for.NETSuite package includes:
  • "/DLL/Net20/KeepAutomation.Barcode.*.dll" includes all dlls built using .net framework 2.0

  • "/DLL/Net40/KeepAutomation.Barcode.*.dll" includes all dlls built using .net framework 4.0

  • "/DLL/SSRS/KeepAutomation.Barcode.*.dll" includes all dlls for SSRS reports

  • "/Developer Guide/Developer-Guide.html" includes guide for developers to use the SDK.

  • "/License Agreement/" includes license agreement document

  • "/Buy Now/" includes link for online license purchase

  • "/barcode/" includes ASP.NET web application

  • "/Sample Data/" includes sample data set for demo code
